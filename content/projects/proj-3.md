# 항공 Aggregator 플랫폼 'Luna' 기획 총괄

## 프로젝트 성격
NDC Aggregator 기반 항공 상품 판매·예약·관리 기능을 제공하는 B2B/B2C용 항공 예약 플랫폼 구축

## 목표
API 연동이 어려운 사용자도 NDC 운임·재고에 접근할 수 있도록, 검색→예약→발권 흐름을 단일 플랫폼에서 처리 가능하도록 재구성

## 1. 프로젝트 개요

### 프로젝트 명
Luna (Aggregator Agency Portal)

### 제공 기능
- NDC Aggregator를 통해 제공되는 항공 상품 판매
- 사용자에게 예약 생성/관리/발권 가능한 기능 제공
- 기존 API 연동이 어려운 사용자도 접근 가능한 게이트웨이 역할

## 2. 서비스 목적

### 🔧 편리한 NDC 예약 생성 & 관리 기능 제공
- API 연동이 불가하거나, 기존 연동 API를 일시적으로 사용하기 어려운 고객의 불편 해소
- 사용자 단에서 직접 NDC 기반 예약 생성/조회/변경 가능

### 🚀 B2B 서비스 확장
- 기존 B2B 고객도 별도 API 연동 없이 NDC 운임 판매/수정 가능
- 제휴사·여행사 대상 확장 가능한 구조 확보

## 3. 대상 사용자
- NDC Aggregator를 통해 항공권 예약이 필요한 일반 사용자
- B2B / B2C 플랫폼, 파트너사, 여행사, 제휴 운영사

## 4. 문제정의
- 공급사별 가격/재고/규정 정보 불일치 → 예약 실패/취소 증가
- 자동 발권 체계 부재 → 수동 처리 과다 & 업무 리드타임 증가
- 환불/변경/재발권 정책 미표준화 → 상담 편차, CS 불만 증가
- 검색 기준(최저가/최단/환승최소) 불명확 → 선택 어려움
