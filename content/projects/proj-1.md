# 회원가입/로그인 간소화

## 서비스 요약
초기 진입 단계에서 강제 로그인 구조로 인해 상품 탐색 → 예약 전환 → 가입 전환으로 이어지는 핵심 퍼널이 끊어지는 문제를 해결하고, 모바일 중심 환경에 맞는 간편 인증/로그인 구조로 재설계한 프로젝트.

## 개발기간
2개월

## 투입인원
기획 1명, 디자인 1명, 개발 1명

## 🚩 문제정의
- 강제 로그인 구조로 상품 탐색 단계 이탈률 증가
- 복잡한 인증/가입 경로로 신규 가입 전환율 저하
- 모바일 중심 사용자 환경 대비 무거운 가입 UI/UX
- 자체회원/SNS회원 분리체계 → 계정 혼선/CS 문의 증가
- SNS 로그인 이용자의 비밀번호 찾기 불가로 불편 지속

## 🎯 목표
- 신규 가입 전환률 상승
- 소셜 로그인 통합 및 계정 혼선 해소
- 모바일 퍼스트 환경 최적화
- 예약 흐름 이탈 최소화 → 서비스 성장에 기여

## 🔄 FLOW 개선

### Before
상품 상세 → 예약 시도 → 로그인/회원가입 강제 → 이탈 발생

### After
상품 상세 → 탐색 가능(로그인 불필요) → 예약 시도 → 간편 로그인/가입 → 예약 완료

## 🧩 핵심 의사결정
- 인증 체계 변경: CI 기반 → 점유 기반 / 보안 유지 + 유지보수 효율화
- 계정체계 변경: 아이디(ID) → 이메일 기반 / SNS 중복 해결
- UI/UX 개선: 전 채널(GNB/예약/모달)에서 일관된 로그인 경험 제공
- 모바일 퍼스트 최적화: 최소 입력/최소 터치 중심

## 📈 성과
- 회원가입 전환률 100% 증가
- 로그인 단계 이탈률 24% 감소

## 🧑‍💻 역할
- 정책 정의, 벤치마킹, 요구사항 정리
- UI/UX 설계 및 페이지 플로우 정의
- 회원 DB/계정체계 설계 및 QA 진행
