# AI 기반 항공 환불 수수료 조회 봇

## 서비스 요약
항공권 환불 규정 복잡성·상담 편차로 인해 발생하던 CS 부담을 완화하고, 내부 상담 프로세스 의존도를 낮추기 위한 AI 기반 환불 수수료 자동 조회 솔루션 구축.

## 개발기간
3개월

## 투입인원
기획 1명, 개발 1명, AI 담당 1명

## 🚩 문제정의
- 항공사별 규정 해석 차이 → 상담사별 응대 편차 발생
- 환불 수수료 확인까지 평균 1~2일 소요 / 대기 불만 증가
- 동일 문의 반복 발생으로 CS 리소스 과소비

## 🔄 FLOW 개선

### Before
고객 1:1 문의 접수 → 상담원 순차 확인 → 늦은 답변

### After
로그인 → 예약 상세 → AI 자동 계산 → 즉시 환불 수수료 안내

## 🧩 핵심 의사결정
- 항공사 규정 기반 응답 구조화 → AI 계산 신뢰성 확보
- 규정 명확/불명확 케이스 분리 → 자동화 가능 범위 제한
- 불확실 시 상담 전환(고객 신뢰/리스크 최소화)
- 항공권 유형/구매시점/조회 타이밍 기반 계산 로직 도입

## 🎯 목표
- 고객 셀프 해결률 증가
- 상담 대기 리드타임 단축
- 신뢰 기반 CX 향상

## 📈 성과
- 환불 수수료 관련 1:1 문의 70% 감소
- AI 응답 정확도 100% (테스트 기준)

## 🧑‍💻 역할
- 항공 규정 데이터 구조화/필드 정의
- 환불 수수료 FLOW 및 계산 로직 설계
- 챗봇 QA 시나리오, 엣지케이스 운영 정책 정립
