# 문진경 이력서 포트폴리오 - 코드 및 내용 정리

## 📋 프로젝트 개요

이 프로젝트는 Next.js 14 기반의 포트폴리오 웹사이트로, Product Owner/Product Manager 이력서를 웹으로 표현한 것입니다.

### 기술 스택
- **프레임워크**: Next.js 14.2.5 (App Router)
- **언어**: TypeScript
- **스타일링**: Tailwind CSS
- **애니메이션**: Framer Motion
- **데이터 관리**: Excel 파일 기반 자동 생성 시스템

---

## 🏗️ 프로젝트 구조

```
app/
├── api/resume/route.ts      # 이력서 PDF 다운로드 API
```
