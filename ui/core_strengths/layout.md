# 레이아웃 가이드

## 그리드 레이아웃

- **데스크톱 (lg 이상)**: 3열 그리드 (`grid-cols-3`)
- **태블릿 (md)**: 2열 그리드 (`grid-cols-2`)
- **모바일**: 1열 그리드 (`grid-cols-1`)

## 간격 및 여백

- 카드 간 간격: `gap-8`
- 섹션 상하 여백: `py-20`
- 카드 내부 패딩: `p-6`
- 섹션 최대 너비: `max-w-6xl`

## 반응형 브레이크포인트

- 모바일: 기본 (< 768px)
- 태블릿: `md:` (≥ 768px)
- 데스크톱: `lg:` (≥ 1024px)

## 애니메이션

- 각 카드는 순차적으로 나타남 (delay: index * 0.1초)
- 페이드인 + 위로 슬라이드 효과
- 호버 시 그림자 효과

## 색상 스키마

- 배경: `bg-white` / `dark:bg-gray-800`
- 테두리: `border-gray-200` / `dark:border-gray-700`
- 제목: `text-gray-900` / `dark:text-white`
- 설명: `text-gray-600` / `dark:text-gray-400`
