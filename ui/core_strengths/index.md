# 핵심 역량

## 섹션 소개

공통 도메인을 기반으로 서비스 전반을 구조화하고, 데이터와 운영 기준 중심으로 문제를 해결하는 PM입니다. 자동화와 예외 처리를 고려해 확장 가능한 제품을 설계합니다.

## 사용 목적

포트폴리오의 "핵심 역량" 섹션에서 사용자의 주요 강점을 카드 형태로 시각적으로 표현합니다. 각 강점은 명확한 제목과 설명으로 구성되어 있으며, Product Manager로서의 핵심 역량을 한눈에 파악할 수 있도록 합니다.

## 컴포넌트 구조

- `StrengthCard.tsx`: 개별 강점 카드 컴포넌트
- `strengths.json`: 강점 데이터 (3개)
- `layout.md`: 레이아웃 가이드
- `labels.md`: 라벨 및 텍스트 가이드
