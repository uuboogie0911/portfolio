## 🚀 사용 방법

1. **엑셀 파일 생성**
   - `data/portfolio.xlsx` 파일을 생성하고 위 구조에 맞춰 작성하세요

2. **데이터 빌드**
   ```bash
   npm run build:data
   ```
   이 명령어를 실행하면 `data/portfolio-generated.ts` 파일이 자동 생성됩니다.

3. **개발 서버 실행**
   ```bash
   npm run dev
   ```

4. **빌드 시 자동 변환**
   ```bash
   npm run build
   ```
   빌드 시 자동으로 엑셀 파일을 읽어서 데이터를 생성합니다.

## 📝 주의사항

- `data/portfolio-generated.ts` 파일은 자동 생성되므로 직접 수정하지 마세요
- 엑셀 파일을 수정한 후에는 `npm run build:data`를 실행해야 변경사항이 반영됩니다
- 개발 중에는 `npm run dev`를 실행하면 자동으로 감지되어 반영됩니다 (Next.js의 파일 감지 기능)

## 🔄 업데이트 프로세스

1. `data/portfolio.xlsx` 파일을 엑셀로 열어서 수정
2. 저장
3. `npm run build:data` 실행 (또는 개발 서버 재시작)
4. 웹사이트에서 확인
